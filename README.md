# Movie Recommendation System

## Overview
This project recommends movies based on user preferences using mathematical similarity techniques.

## Features
- Takes user input for preferences (Action, Comedy, Suspense)
- Calculates similarity between user and movies
- Recommends the closest matching movie

## Tech Used
- Python
- NumPy

## Concepts Used
- Vector representation
- Euclidean distance
- Cosine similarity (initial approach)

## How It Works
User preferences and movies are represented as vectors.  
The system calculates distance between vectors and recommends the closest match.

## Future Improvements
- Add multiple users
